Current Courses

Course Name Description
CSC5750: Web Technology This course discusses the Internet and the World Wide Web. It covers techniques for authoring static and dynamic content for the web.
CSC4320: systems adminstration This course introduces students to Linux operations and system administration through a unified virtual machine installation. Students will gain experience working with differing hardware configurations while testing interoperability between Linux and Windows. They will learn about and practice Linux administration with helpful in-text features such as learning objectives, key terms, and self-assessment through review questions, hands-on activities, and more.
CSC3400: Human Computer This course is an introductory course on human-computer interaction. It does not presuppose any earlier knowledge of human-computer interaction, computer science or psychology. The class covers two broad categories of topics within human-computer interaction: (a) the principles and characteristics of the interaction between humans and computers and (b) the techniques for designing and evaluating user-centered systems. .
CSC4190: Computer Netwotk With no prior networking knowledge required, this course provides a full overview of computer networking. A top-to-bottom approach is leveraged to understand essential network concepts and protocols, from high-level applications to low-level operational functions, including HTTP, TCP/IP, routing, LAN, wireless, etc. Moreover, you will learn socket programming for data communication. Offered every fall/winter (or yearly if offering in summer too) Restriction(s): Enrollment is limited to BSIT students only.

In addition, statistical analysis of the system performance such as the mean time between failures (MTBF); and the average time spent in recovering from an outage, usually called the mean time to recover (MTTR) will be covered.
